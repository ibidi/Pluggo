# Contributing

Thanks for contributing to Pluggo.

## Development

- macOS 13+
- Xcode Command Line Tools / Xcode
- Swift 5.9+

Run locally:

```bash
swift run
```

Build release app bundle:

```bash
./build.sh --no-dmg
```

## Pull Requests

- Keep PRs focused.
- Explain user-visible changes.
- Include screenshots for UI changes.
- Mention permission-related behavior changes clearly.

## Reporting Bugs

Please include:

- macOS version
- provider used (Groq/OpenAI/LibreTranslate)
- source/target language
- logs from Pluggo (`Log / Hata -> Kopyala`)
